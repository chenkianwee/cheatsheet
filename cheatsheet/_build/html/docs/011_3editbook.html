Edit the Book

10.3. Edit the Book#

  1. Use a text editor to edit the files. I am using atom. Download Atom here

  2. Open Atom and add the mynewbook folder. Figure 10.4

  3. Once added you will be able to see the structure of the book in Figure 10.5.

  4. To change the title of the book, go to the “_config.yml” file. Change the title of the book from “My sample book” -> “My notebook”. Change the author to your name (Figure 10.6). Save the change. Build the book with the command from the 4th Step of Create a Book Template with Jupyter-book.

    a. The result of the change (Figure 10.7).

  5. Next I will make changes to the _toc.yml. This file controls the table of content on the left side of the screen. I will add a new file and some content to the file.

    a. I add a new file called newcontent as shown in Figure 10.8. Remember to save the change and then build the book again.

    b. With Atom add a file called “newcontent.md” as shown in Figure 10.9. Add in the title “# My new content”.

    c. Build the book. You will be able to see the change as shown in Figure 10.10
